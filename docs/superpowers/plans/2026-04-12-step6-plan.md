# M5 Step 6 Permalink Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver the `/~:username/:year/:month/:day/:slug` permalink so published posts render while drafts stay author-only.

**Architecture:** Leverage the existing `PostStorage::get_post_by_permalink` call inside a new `get_post` server fn, add Leptos routing/component logic that consumes it, and validate the end-to-end path with Playwright.

**Tech Stack:** Rust (Axum/Leptos), sqlx-backed storage, Playwright + TypeScript, cargo tooling as in repo.

- [ ] **Step 3: Implement `get_post` in `web/src/posts.rs`**

Replace the placeholder `get_post` with the auth-aware version:

```rust
/// Retrieves a post by its permalink.
#[server(endpoint = "/get_post")]
    username: String,
    year: i32,
    month: u32,
    day: u32,
    slug: String,
) -> Result<PostResponse, ServerFnError> {
    #[cfg(feature = "ssr")]
    {
        use common::{slug::Slug, username::Username};

        NaiveDate::from_ymd_opt(year, month, day)
            .ok_or_else(|| ServerFnError::new("Invalid permalink"))?;

        let username_parsed = username
            .parse::<Username>()
            .map_err(|e| ServerFnError::new(e.to_string()))?;
        let slug_parsed = slug
            .parse::<Slug>()
            .map_err(|e| ServerFnError::new(e.to_string()))?;

        let state = expect_context::<Arc<AppState>>();
        let post = state
            .posts
            .get_post_by_permalink(&username_parsed, year, month, day, &slug_parsed)
            .await
            .map_err(|e| ServerFnError::new(e.to_string()))?
            .ok_or_else(|| ServerFnError::new("Post not found"))?;

        Ok(PostResponse {
            post_id: post.post_id,
            username: username_parsed.to_string(),
            title: post.title,
            slug: post.slug.to_string(),
            body: post.body,
            format: post.format.to_string(),
            rendered_html: post.rendered_html,
            created_at: post.created_at.to_rfc3339(),
            published_at: post.published_at.map(|t| t.to_rfc3339()),
        })
    }
    #[cfg(not(feature = "ssr"))]
    {
        let _ = (username, year, month, day, slug);
        Err(ServerFnError::new("Not implemented"))
    }
}
```

- Add an adjacent code block describing the new `get_post_preview(post_id: i64)` server fn: it requires auth, fetches via `AppState.posts.get_post_by_id`, checks `post.user_id == auth.user_id`, rejects soft-deleted rows, and reuses `PostResponse`. (All preview/permalink URLs still come from `CreatePostResult`; the preview endpoint just returns post content.)

- [ ] **Step 1: Implement `PostPage` with param validation, draft badge, and home link**

Replace the placeholder component in `web/src/pages/posts.rs` with:

```rust
use leptos_router::hooks::use_params_map;

#[component]
    let params = use_params_map();
    let post = Resource::new(
        move || (),
        move |_| {
            let params = params.get();
            let raw_username = params.get("username").cloned().unwrap_or_default();
            let username = raw_username.trim_start_matches('~').to_string();
            let slug = params.get("slug").cloned().unwrap_or_default();
            let year = params.get("year").and_then(|v| v.parse::<i32>().ok());
            let month = params.get("month").and_then(|v| v.parse::<u32>().ok());
            let day = params.get("day").and_then(|v| v.parse::<u32>().ok());

            async move {
                if username.is_empty()
                    || slug.is_empty()
                    || year.unwrap_or(0) < 1
                    || month.unwrap_or(0) < 1
                    || day.unwrap_or(0) < 1
                {
                    return Err(ServerFnError::new("Invalid permalink"));
                }
                get_post(
                    username,
                    year.unwrap(),
                    month.unwrap(),
                    day.unwrap(),
                    slug,
                )
                .await
            }
        },
    );

    view! {
        <Suspense fallback=|| view! { <p>"Loading..."</p> }>
            {move || Suspend::new(async move {
                match post.await {
                    Ok(post) => view! {
                        <article>
                            <h1>{post.title.clone()}</h1>
                            <p class="metadata">
                                "By "
                                <a href=format!("/~{}/", post.username)>{post.username.clone()}</a>
                                " on "
                                {post
                                    .published_at
                                    .clone()
                                    .unwrap_or_else(|| post.created_at.clone())}
                            </p>
                            <div class="content" inner_html=post.rendered_html.clone()></div>
                        </article>
                        <p><a href="/">"Home"</a></p>
                    }
                    .into_any(),
                    Err(err) => view! { <p class="error">{err.to_string()}</p> }.into_any(),
                }
            })}
        </Suspense>
    }
}
```
